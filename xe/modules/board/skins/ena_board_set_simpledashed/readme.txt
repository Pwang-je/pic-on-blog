* 설명
이 스킨은 개인홈페이지에 맞춰 만든 게시판입니다. 그러므로 커뮤니티 등 용도 외의 사용에는 적합하지 않을 수 있습니다.

**이 스킨은 xe 코어 1.7.10에서 테스트 되었습니다.
***이 스킨은 다국어 지원을 고려하지 않았습니다.
****스킨의 원 제작자 및 개작자(이나)는 스킨에 대한 보증 및 as 의무가 없습니다. 또한 스킨과 관련한 질문에 대해서도 답변 의무가 없습니다. 

* 적용 방법
파일을 받아 압축을 풀고 폴더(ena_board_set_simpledashed)를 xe 설치 폴더/modules/board/skins 에 업로드한 다음에 게시판 정보에서 적용하면 됩니다. 이 때, ena_board_set_simpledashed **폴더 바로 안에** html, css, js 등의 파일이 존재해야 합니다.

* 설정 안내
게시판 관리 - 스킨 관리에서 여러가지 설정을 이용할 수 있습니다.
	#게시판 유형 : 게시판의 유형을 선택할 수 있습니다. 용도별, 취향별로 다르게 설정해보세요.
	#포인트 색상 : 스킨 포인트 부분의 색상을 지정할 수 있는 기능. 예시) #000000, black, rgba(0,0,0,0.5) 만일 적지 않을 시에는 css에서 기본 설정된 색이 나옴.
	#글쓰기 토글 사용 : list_read를 사용하는 스킨 유형에만 적용(일기장 제외). 예:토글 사용, 아니오:토글을 사용하지 않고 별도의 글쓰기 페이지로 이동. 토글 사용시 글쓰기가 동작하지 않는 경우에 아니오로 변경하세요.
	#목록에서 카테고리 출력 : 일반형에서 카테고리 표시 형식을 바꿀 수 있습니다. 예1은 카테고리 길이가 다양하고 미분류도 자주 사용되는 경우, 예2는 카테고리가 짧고 미분류는 잘 사용되지 않는 경우에 선택하는 것을 추천합니다.
	#목록 스타일 : 일반형을 일반 게시판으로 쓸 것인지, 텍스트 링크 게시판으로 사용할 것인지를 선택할 수 있습니다.
		텍스트 링크 : 배너 없는 링크 게시판을 만들 때 사용. 사용자 정의(이하 확장변수) link_url을 필요로 합니다. 비밀글은 숨겨주지 않습니다.
		link_url : 링크 주소 / 형식은 url 형식, 필수 항목은 y로 하세요.
	#글쓴이 닉네임 표시 : 일기장 본문에서 글쓴이의 닉네임을 표시할 것인지를 선택할 수 있습니다. 일반적으로 공동 일기장을 운영하지 않는 이상 미표시로 설정하면 됩니다.
	#본문 카테고리 표시 : 일기장 본문에서 카테고리 표시 기능.
	#댓글 표시 : 댓글쓰기 폼만 숨길지, 일반 메모장에서 댓글을 토글로 숨길지를 선택할 수 있습니다. 
	#목록 썸네일 넓이-높이 : 갤러리에서 목록의 썸네일 크기를 정할 수 있습니다. 다양하게 정할 수 있으나 넓이와 높이를 같게 하는 것을 추천합니다. 기본값은 200(px)입니다. 단위는 px이나 입력할 때에는 숫자만 입력해야 합니다.
	#카테고리 표시 : 배너 링크에서 카테고리 표시 기능. 제목 옆에 표시됩니다.
	#배너 링크 : 다음의 확장변수가 필요합니다.
		link_url : 링크 주소 / 형식은 url 형식, 필수 항목은 y로 하세요.
		link_img : 배너 / 형식은 url 형식, 필수 항목은 y로 하세요.(직링크 방식입니다.) 90px*30px를 권장하지만 더 길거나 짧은 배너도 지원합니다. 링크 이미지 크기는 되도록 비슷하게 맞춰주세요.
		link_explain : 설명 / 형식은 한줄 입력칸(text), 필수 항목은 n으로 하세요. 공지를 포함해 모든 글에서 사용 가능합니다. 제목 밑에 표시됩니다.
		- 비밀글은 숨겨줍니다. 목록에는 비밀글이 아닌 글과 권한이 있는 글만 나타납니다.
